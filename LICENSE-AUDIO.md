## How they were made

Both cues are mathematically synthesized 16-bit PCM WAV files (mono,
22 050 Hz). The exact frequencies, envelopes, and durations are defined in
[`scripts/generate-cues.mjs`](scripts/generate-cues.mjs):

- **`enter-reserve.wav`** — 250 ms · 880 Hz fundamental + 1320 Hz overtone
  (perfect fifth) · ADSR envelope 5 / 40 / 50 / 180 ms · peak ~ -11 dBFS.
  Character: a soft, gentle bell-like ding.
- **`exhausted.wav`** — 300 ms · two 120 ms pulses of 440 Hz with mild
  harmonic content (1.5×, 2.0× partials) separated by a 60 ms gap · peak
  ~ -11 dBFS. Character: a quick, distinct "two-beep" alert.

Because the files are the byte-for-byte deterministic output of running a
short script over public mathematical operations (sine waves, linear
envelopes), they contain no expression eligible for copyright. They are
released into the public domain under the **Creative Commons Zero (CC0)**
dedication and may be used, modified, distributed, and re-licensed without
restriction or attribution.

## Re-generating the assets

If you want to tweak the tones, edit `scripts/generate-cues.mjs` and run:

```powershell
node scripts/generate-cues.mjs
```

The script overwrites the WAV files in place. Commit the updated files
along with any changes to the script and bump this file's "Date added"
column accordingly.

## Replacing with third-party royalty-free assets

If you would prefer pre-recorded sound effects from a curated library,
replace the WAV files with assets from any of the approved royalty-free
sources documented in [`specs/002-turn-sounds/research.md`](specs/002-turn-sounds/research.md)
(Pixabay Sound Effects · freesound.org filtered to CC0 · Mixkit) and add a
new row to the inventory table above with the source URL, license, and
author for each replacement file. The application code accepts both `.wav`
and other common formats — no code changes are required as long as the
extensions in [`assets/js/audio.js`](assets/js/audio.js) match.
